# 🐍 Python Retail & Sales Data Analysis Project  
*A Data Technician Bootcamp Project*

This project showcases my ability to use **Python and pandas** to explore, clean, manipulate, and analyse retail and sales datasets. It highlights the core data analysis skills I developed during my Data Technician bootcamp, including filtering data, subsetting with `.loc[]` and `.iloc[]`, performing transformations, and visualising insights using Python’s plotting libraries.

---

## 📊 Project Overview
The goal of this project was to work with real retail and sales data to uncover trends, identify patterns, and prepare datasets for further analysis. Using pandas and basic visualisation tools, I built a workflow that mirrors real‑world data analysis tasks: loading data, cleaning it, transforming it, and presenting insights clearly.

---

## 🧠 Key Skills Demonstrated

### 🧮 Data Filtering & Subsetting  
- Selecting rows and columns using **`.loc[]`** (label‑based) and **`.iloc[]`** (position‑based)  
- Filtering data using conditions (e.g., sales above a threshold, specific product categories)  
- Creating focused subsets for deeper analysis  

These techniques allowed me to isolate meaningful segments of the dataset and explore them in detail.

---

### 🔧 Data Cleaning & Manipulation with pandas  
Used a range of pandas methods to prepare and transform the data:

- **`dropna()`** — removing missing values  
- **`fillna()`** — replacing missing values with defaults or calculated values  
- **`sort_values()`** — ordering data by revenue, date, or product performance  
- **`groupby()`** — aggregating sales by category, store, or month  
- **`value_counts()`**, **`rename()`**, **`astype()`**, and other transformation methods  

These operations helped clean the dataset and generate meaningful metrics for analysis.

---

### 📈 Data Visualisation  
Created clear and informative charts using:

- pandas’ built‑in `.plot()`  
- **matplotlib** for more customised visuals  

Visualisations included:

- Line charts for monthly sales trends  
- Bar charts for product and category performance  
- Pie charts for distribution analysis  

These visuals supported data storytelling and helped communicate insights effectively.

---

## 🧩 How These Skills Came Together
By combining pandas operations with visualisation tools, I was able to:

- Clean and prepare raw retail datasets  
- Analyse sales patterns across products, stores, and time periods  
- Identify top‑performing categories and seasonal trends  
- Present insights through clear, data‑driven visuals  

This project strengthened my ability to work end‑to‑end with data using Python.
